# Contribution Guidelines

Thank you for your interest in contributing to the **Awesome Developer Roadmaps** project! Your help in curating the best learning paths for the community is highly appreciated.

## How to Contribute

You can contribute by:
- Adding a new, high-quality roadmap
- Fixing broken links
- Updating the "Last Updated" year to reflect the most current state
- Improving descriptions and difficulty estimates

## Adding a New Roadmap

To add a new roadmap, please follow these steps:

1. **Check for duplicates**: Ensure the roadmap (or a very similar one) isn't already on the list.
2. **Format**: All roadmaps must be added to the markdown tables in the appropriate section.
   Use the following format for your entry:
   ```markdown
   | [Roadmap Name](URL) | 🟢 Beginner | 2–4 Months | 🌐 Website | 2024 | A short, informative description. |
   ```
3. **Columns Explained**: 
   - `Roadmap`: The name of the resource, hyperlinked.
   - `Difficulty`: Use emojis (🟢 Beginner, 🟡 Intermediate, 🔴 Advanced). Combinations are allowed (e.g., 🟢 Beginner → 🔴 Advanced).
   - `Estimated Time`: A realistic time estimate (e.g., `2–4 Months`). If unknown, use `—`.
   - `Resource Type`: Use emojis:
     - 🌐 Website
     - 📂 GitHub
     - 📄 PDF
     - 🎥 Video Course
     - 📚 Documentation
     - 🎓 Course
     - 🛠 Interactive
   - `Last Updated`: The most recent year the source was updated (e.g., `2024`). If unknown, use `—`.
   - `Description`: A concise description of the roadmap's contents.
4. **Placement**: Add your entry under the most appropriate section (e.g., *Web Development*, *Mobile Development*). Maintain alphabetical or logical order within sections.

## Pull Request Process

1. Fork the repository and create your branch from `main`.
2. Make your changes in your branch.
3. Ensure your markdown tables are aligned properly.
4. Submit a Pull Request with a clear title and description of your changes.

If you aren't sure how to make a Pull Request, you can also open an **Issue** with the link and details, and we can add it for you.

Thank you for making this awesome list better!
