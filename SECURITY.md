# Security Policy

## Supported Versions

Since this repository primarily hosts documentation and links (markdown files), we do not have software "versions" in the traditional sense that require security patches. We support the `main` branch.

## Reporting a Vulnerability

If you discover a malicious link, a hijacked domain, or any other security issue within the repository's contents, please report it immediately by opening an Issue labeled `security`. 

If the issue is highly sensitive, please reach out to the maintainers directly via their GitHub profiles.

We will attempt to remove any malicious links as soon as they are verified.
