# Frequently Asked Questions (FAQ)

### What is a learning roadmap?
A learning roadmap is a strategic, step-by-step guide that outlines the technologies, concepts, and tools you need to learn to achieve a specific career goal (e.g., becoming a Frontend Developer). 

### Do I need to learn everything on a roadmap?
**Absolutely not.** Roadmaps are exhaustive by design to show you the entire landscape. You only need to learn the core concepts and *one* tool in each category. For example, if a roadmap lists React, Vue, and Angular, you only need to pick one.

### How do I escape "Tutorial Hell"?
Tutorial hell is when you endlessly watch tutorials but cannot build anything on your own. To escape:
1. Stop watching videos passively.
2. Follow a tutorial, but then attempt to add a new feature to the project without help.
3. Start a blank project and try to build something from scratch. Read documentation when you get stuck instead of looking for a video.

### Do I need a Computer Science degree?
No. The tech industry heavily values skills over degrees. A strong portfolio of projects and open-source contributions can often outweigh a degree, especially in web and mobile development. However, for specialized fields like AI, Machine Learning, or Systems Programming, a degree is highly beneficial.

### Is AI going to replace developers?
AI (like GitHub Copilot and ChatGPT) are incredibly powerful tools that *augment* developers, making them faster and more efficient. They are not currently capable of autonomously gathering requirements, designing complex architectures, and maintaining massive codebases. Learn to use AI as a tool to write boilerplate code so you can focus on architecture and problem solving.
