# Interview Preparation

Preparing for technical interviews requires a different skill set than day-to-day software engineering. Here are the curated resources for interview prep.

## Data Structures and Algorithms (DSA)

- **[Coding Interview University](https://github.com/jwasham/coding-interview-university):** The most comprehensive study guide.
- **[NeetCode](https://neetcode.io/):** A highly structured way to practice LeetCode problems (Blind 75 and NeetCode 150).
- **[AlgoExpert](https://www.algoexpert.io/):** (Paid) Excellent video explanations for algorithmic problems.

## System Design

- **[System Design Primer](https://github.com/donnemartin/system-design-primer):** The best free resource.
- **[ByteByteGo](https://bytebytego.com/):** Created by Alex Xu, author of "System Design Interview". Highly recommended.
- **[Grokking the System Design Interview](https://www.educative.io/courses/grokking-the-system-design-interview):** (Paid) A classic interactive course.

## Behavioral Interviews

Don't neglect the behavioral round. Use the **STAR Method** (Situation, Task, Action, Result) to answer questions like "Tell me about a time you had a conflict with a coworker."
