# Certifications Guide

In software engineering, certificates are generally less valuable than a strong portfolio. However, in specific fields like **Cloud**, **DevOps**, and **Cybersecurity**, they are highly respected and sometimes required.

## Cloud Certifications

### Amazon Web Services (AWS)
1. **AWS Certified Cloud Practitioner:** Entry-level, good for non-technical roles.
2. **AWS Certified Solutions Architect – Associate:** The most valuable foundational cloud certification. Highly recommended.
3. **AWS Certified Developer – Associate:** Focused more on using AWS SDKs and CI/CD.

### Microsoft Azure
1. **AZ-900: Microsoft Azure Fundamentals:** Entry-level.
2. **AZ-204: Developing Solutions for Microsoft Azure:** Developer focused.
3. **AZ-104: Microsoft Azure Administrator:** Ops focused.

## Cybersecurity Certifications

1. **CompTIA Security+:** The industry standard entry-level security certification.
2. **CEH (Certified Ethical Hacker):** Recognized, though sometimes criticized for being too theoretical.
3. **OSCP (Offensive Security Certified Professional):** The gold standard for penetration testers. It is a grueling, highly practical exam.

## IT Fundamentals

1. **CompTIA A+:** Good if you are starting in IT helpdesk before moving to development/ops.
2. **CompTIA Network+:** Excellent foundation for understanding how the internet actually works.
